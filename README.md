# ContaExpert MD — Landing Contabilitate

Landing page pentru Google Ads (Contabil / Contabilitate).  
**RO / RU / EN** · Framer Motion · SEO · Schema.org · OG

## Start local

```bash
npm install
npm run dev
```

Deschide http://localhost:3000

## Env

Copiază `.env.example` → `.env.local`:

```
NEXT_PUBLIC_SITE_URL=https://your-project.vercel.app
NEXT_PUBLIC_FORM_URL=https://formspree.io/f/YOUR_ID
```

## Deploy Vercel

1. Push pe Git (GitHub Desktop)
2. [vercel.com](https://vercel.com) → Import repo
3. Framework: Next.js (auto)
4. Adaugă env variables
5. Deploy
